jQuery Bar Rating

Minimal, light-weight jQuery ratings.

jQuery Bar Rating Plugin works by transforming a standard select field into a rating widget. The rating widget can be flexibly styled with CSS. Take a look at a few examples.

How to use
  1. Get the plugin from GitHub or install with Bower: 
    
bower install jquery-bar-rating
  2. Create a select field: 
    
<select id="example">
  <option value="1">1</option>
  <option value="2">2</option>
  <option value="3">3</option>
  <option value="4">4</option>
  <option value="5">5</option>
</select>

  3. Include and call the plugin (after jQuery v1.7.2+) 

    
<script src="jquery.barrating.min.js"></script>
<script type="text/javascript">
   $(function() {
      $('#example').barrating();
   });
</script>
  4. The select field will be wrapped in a div with a class `.br-wrapper`. You can change the div's class by specifying the `wrapperClass` setting when calling the plugin. The `wrapperClass` setting is useful if you plan to have a few differently styled ratings coexist together. Expected result: 
    
<div class="br-wrapper">
  <select id="example">
    <option value="1">1</option>
    <option value="2">2</option>
    <option value="3">3</option>
    <option value="4">4</option>
    <option value="5">5</option>
  </select>
</div>

  5. By default there are no styles applied to the wrapper div. A few examples of CSS styling are provided that you can use as a starting point to customize the plugin to your needs.

Configuration

initialRating: null
Specify initial rating by passing select field's option value. If null, the plugin will try to set the initial rating by finding an option with a `selected` attribute.

showValues: false
If set to true, rating values will be displayed on the bars.

showSelectedRating: true
If set to true, user selected rating will be displayed next to the widget.

reverse: false
If set to true, the ratings will be reversed.

readonly: false
If set to true, the ratings will be read-only.

fastClicks: true
Remove 300ms click delay on touch devices.

wrapperClass: 'br-wrapper'
Specify wrapper div's class. Useful in cases when you want to have a few differently styled ratings coexist together.

Methods

$('select').barrating('show');
Shows the rating widget.

$('select').barrating('set', value);
Sets the value of rating widget.

$('select').barrating('clear');
Clears the rating.

$('select').barrating('destroy');
Destroys the rating widget.

Callbacks

onSelect:function(value, text)
Fired when a rating is selected.

onClear:function(value, text)
Fired when a rating is cleared.

onDestroy:function(value, text)
Fired when a rating is destroyed.
